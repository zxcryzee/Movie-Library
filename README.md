# Movie Library - Личная кинотека

**Автор:** Помелов Ярослав

## Описание

Десктопное приложение для ведения личной коллекции фильмов.  
Позволяет добавлять, просматривать, фильтровать и сохранять фильмы в формате JSON.

### Возможности

- Добавление фильмов (название, жанр, год, рейтинг)
- Валидация ввода (год – число, рейтинг 0-10)
- Фильтрация по жанру и году
- Автосохранение в JSON
- Табличный просмотр

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `movie_library.py`
3. Запустите в терминале:

```bash
python movie_library.py
